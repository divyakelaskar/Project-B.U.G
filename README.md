# Project B.U.G
A browser based 3D FPS game.


## 🛠️ Built with  :  
- Unity
- C#
- HTML


## ⚙️ Controls  : 

🏃 Movement : WASD / arrow keys

🔫 Shoot : Left mouse button

🎯 Aim down sight : Right mouse button

🔄 Switch weapons : Number keys / Q and E keys / Middle mouse button

🦘 Jump : Space key

🙇 Crouch : C key
